Browser Test Results Viewer

How to Run Tests

  1. Run ./tests/run-browser-tests.sh from the project root
  2. Open one of these URLs in your browser:
    • http://localhost:8080/tests/browser/test-runner.html
    • http://localhost:8080/tests/browser/test-runner-modules.html
  3. The tests will run automatically and show results in the browser

Common Test Issues & Solutions

Issue: Module loading errors

If you see errors like Failed to load module from URL..., check that:

  • You're running the tests through the HTTP server (not opening the HTML file directly)
  • The path to the component modules is correct
  • The server is running from the project root directory
  • Try both test runners, as they use different module loading approaches

Issue: Function not defined errors

If you see errors like window.resetTestDOM is not a function, check that:

  • The helper functions are defined before the tests run
  • The test file is loading correctly
  • Try refreshing the page after a few seconds (module loading may take time)
  • Check the browser console for more detailed error messages

Issue: Module loading timing problems

If tests run before modules are fully loaded:

  • Open the browser console to see loading progress messages
  • Look for "✅ Components loaded, running tests..." message
  • The tests now have retry logic to wait for components to load
  • If the test still fails, try reloading the page
